# Global Democracy Analysis 🌍

This project performs exploratory data analysis on country statistics and democracy scores using Python and Pandas.

## 🚀 Technologies Used
- Python
- Pandas
- NumPy
- Jupyter Notebook

## 📊 Features
- Data cleaning
- Sorting and filtering
- Country-wise analysis
- Democracy score analysis
- Finding top and bottom ranked countries

## 📁 Project Structure
```text
country-data-analysis/
│
├── data_capston.ipynb
├── dataset.csv
└── README.md
```

## ▶️ How to Run
1. Clone the repository
2. Open the notebook in Jupyter Notebook or VS Code
